## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14+)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/krushna001m/Healthcare-Dashboard.git
   cd Healthcare-Dashboard
````

2. **Install dependencies**

   ```bash
   npm install
   # or
   ```
